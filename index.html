Mode d'emploi

Notes

  • Utilisation des notations anglaises (A-G)
  • Note par défaut = noire ; pour une croche utiliser la lettre minuscule
  • Silence : "."
  • Tous les autres caractères sont ignorés (dont le retour à la ligne)

Symboles

  • L'ajout d'un symbole après une note permet de la modifier ; il est possible de cumuler des symboles différents.
  • "#" : dièse (bémol non disponible)
  • "-" : double la note (noire > blanche ; croche > noire)
  • "," et "^" : octave du dessous/dessus
  • "|" : aide pour la mesure des temps (dans la console navigateur)

Modifications permanentes

  • "S" + symbole : permet de changer un paramètre par défaut jusqu'à la fin (pour l'instant, uniquement les octaves)
